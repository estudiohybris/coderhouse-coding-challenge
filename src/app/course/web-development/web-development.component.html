¿QUÉ VAS A APRENDER?

Durante el curso aprenderás a crear sitios web partiendo desde un prototipo en papel, utilizando estos dos lenguajes HTML y CSS. Estos lenguajes contemplan aspectos como: etiquetas principales, selectores, colores, fuentes, medidas, fondos, bordes, animaciones, herramientas de posicionamiento en Google, aspectos de seguridad, hosting, conceptos básicos de jQuery.
